《紫川》

作者：老猪
简述：一部读了很多遍的小说，用略微幽默诙谐的语言，细腻的文笔，特色鲜明的人物刻画，篇幅宏大，剧情跌宕起伏，场面恢弘壮阔，可读性相当之高，称得上是一部史诗大作。
剧情描述从第一次远东战争开始，到远东起义，西南统领，澜沧江会战，卫国战争，一直到三杰之乱，帝国统一。剧情安排跌宕起伏，人物的刻画又细腻饱满。精明能干重情重义的紫川秀，勇猛刚毅的斯特林，果断毒辣的帝林，完美无瑕的流枫霜。。。悲壮，凄惨，充斥着兄弟情谊，儿女之情，人类大义。。。

github pages 配置

简单介绍下所谓的部署到github上的原理：

  1. 在Github上创建的那个特别的repository（name.github.io）一个最大的特点就是其master中的html静态文件，可以通过链接http://name.github.io来直接访问
  2. hexo -g 会生成一个静态网站（第一次会生成一个public目录），这个静态文件可以直接访问
  3. 需要将hexo生成的静态网站，提交(git commit)到github上

部署的步骤如下：
